# Women's Day Interactive Card 🌸

A beautifully designed, interactive web-based greeting card created to celebrate Women's Day.

## 🌟 Features
- **Interactive Narrative**: A sequence of screens that guide the user through a personalized greeting.
- **Dynamic Animations**: Custom CSS animations including floating hearts, flower decorations, and interactive elements.
- **Photo Reveal**: A hidden photo gallery revealed through interaction.
- **Modern Design**: Responsive, modern aesthetic with custom fonts (Dancing Script, Caveat) and a cohesive color palette.
- **Celebratory Finale**: A beautiful final screen with confetti and personalized photos to conclude the experience.

## 📁 Project Structure

This project has been systematically organized for better maintainability:

```text
womens_day_card/
│
├── html/
│   └── index.html    # Main HTML structure and content
│
├── css/
│   └── style.css     # CSS styles, animations, and layout
│
├── js/
│   └── script.js     # Interactivity and screen transitions
│
└── img/
    ├── image_1.jpg   # Photo gallery image 1
    └── image_2.jpg   # Photo gallery image 2
```

## 🚀 How to Use
Simply open `html/index.html` in any modern web browser to view and interact with the card! No server or build process is required.
